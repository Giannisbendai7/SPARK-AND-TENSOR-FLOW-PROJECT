# SPARK-aAND-TENSOO-FLOW-PROJECT
This project focuses on building a data preprocessing and feature engineering pipeline using PySpark.

The goal is to demonstrate how large-scale data can be transformed and prepared for machine learning models using distributed computing techniques.

The project uses PySpark’s VectorAssembler to combine multiple input features into a single feature vector, enabling efficient processing for downstream analytics and ML tasks.

Tech stack: PySpark, Python, Spark DataFrame API

Key steps:

Data loading and preprocessing with PySpark
Feature selection and transformation
Feature engineering using VectorAssembler
Preparing dataset for machine learning pipelines
Scalable data processing using Spark framework
